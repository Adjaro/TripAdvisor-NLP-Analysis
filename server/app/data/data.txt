Welcome to the TripAdvisor NLP Analysis App!

This application allows you to scrape restaurant reviews from TripAdvisor, analyze the data, and visualize the results.

Features:
- Scrape restaurant details and reviews
- Analyze review sentiments
- Visualize data insights

Instructions:
1. Enter the TripAdvisor URL of the restaurant you want to scrape.
2. Click the "Scrape" button to start the scraping process.
3. View the scraped data and analysis results.

Thank you for using the app!
